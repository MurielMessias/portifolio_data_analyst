# Assets de Imagens

## Como Adicionar Imagens ao Modal

1. **Salve suas imagens** nesta pasta com os nomes apropriados
2. **Formatos recomendados**: PNG, JPG, WebP
3. **Tamanho ideal**: Máximo 800px de largura para melhor performance

## Dicas

- Use imagens com fundo transparente (PNG) para melhor integração
