# Portfolio Muriel Ramos Messias

Portfólio profissional desenvolvido com React, TypeScript e Tailwind CSS para apresentação de projetos de análise de dados.

## Tecnologias

- **Frontend**: React 18, TypeScript, Vite
- **Estilização**: Tailwind CSS
- **Ícones**: Lucide React
- **Deploy**: GitHub Pages

## Projetos em Destaque

1. **Análise Exploratória de Ecossistemas Unicórnios (2022)**
   - Python, Pandas, Matplotlib, Seaborn
   - EDA completa sobre dataset de unicórnios globais

2. **Dashboard de Performance Comercial (2022-2024)**
   - Power BI, DAX, Power Query
   - Ecossistema de Business Intelligence

3. **Análise de Desempenho Acadêmico (Students Performance)**
   - Python, Pandas, NumPy, Matplotlib
   - Análise de fatores socioeconômicos no desempenho estudantil

## Desenvolvimento

### Instalação
```bash
npm install
```

### Desenvolvimento Local
```bash
npm run dev
```

### Build para Produção
```bash
npm run build
```

### Deploy no GitHub Pages
```bash
npm run deploy
```
